# Cat Modeling Agency App

Full-stack web application for managing a fictional cat modeling agency, including frontend, backend, and an in-progress API layer for extended functionality.

---

## 🚀 Features

- View and manage cat profiles
- Create, update, and delete cat records
- Backend API handling for cat data
- Multi-service architecture (frontend, backend, API)
- JSON-based data storage (development stage)
- Scalable structure for future features like bookings and analytics

---

## 🧰 Technologies Used

**Frontend:**
- Next.js / React
- JavaScript / TypeScript
- CSS / Tailwind

**Backend:**
- Node.js
- Express.js

**Data Layer:**
- JSON file storage
